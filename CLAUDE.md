# 我的数字视觉档案 — CLAUDE.md

## 项目定位

一份以**苏联构成主义海报**为视觉语言的个人数字档案。

不是作品集。不是简历站。不是程序员主页。

是一组可以翻阅的海报。每一屏都是一张构成主义海报。

### 核心视觉精神

- 对角线统治一切 —— 没有水平居中的标题，没有规整的卡片网格
- 文字就是图形 —— 中文字号大到溢出屏幕，西文用作结构装饰
- 红黑白为主，底色为暖纸米棕 —— 像旧画册般温润的纸质感
- 照片蒙太奇 —— 图片被裁剪、旋转、叠加，像 Rodchenko 用剪刀和浆糊拼出来
- 几何构件 —— 圆、线、条、三角不是装饰，是建筑结构
- 工业感 —— 粗粝、直接、有力。不是苹果店，是印刷厂。

### 参考

| 设计师 | 学什么 |
|--------|--------|
| Rodchenko | 对角线构图、摄影蒙太奇、大字报式排版、红黑白 |
| El Lissitzky | Proun 悬浮几何、空间深度感、建筑化构图 |
| 1920s 苏联电影海报 | 激进不对称、文字与图像同等尺度的对抗 |
| 构成主义 propaganda | 紧迫感、命令式语气、一张海报只讲一件事 |

## 配色

### 暖纸调色板（实际采用）

底色采用暖纸米棕系，替代纯白——像翻阅旧画册一样的温润纸感。

```
红色 — #D10000 (主), #FF2200 (亮红), #8B0000 (深红)
暖黑 — #0D0D0D (纯黑), #3D3228 (暖棕黑)
暖纸 — #E8DCC8 (纸黄底 bg), #D4C4A8 (纸深 bg-muted), #F5EDE0 (纸白 paper-cream)
灰色 — #9A8B78 (中灰 mid), #C9B89A (浅灰 light), #B0B0B0 (冷灰,少用)
```

### CSS 变量映射

| 变量 | 颜色 | 用途 |
|------|------|------|
| `--bg` | #E8DCC8 | 页面背景 |
| `--bg-muted` | #D4C4A8 | 照片占位 / 次要背景 |
| `--paper-cream` | #F5EDE0 | 暗底上的亮文字 |
| `--fg` | #0D0D0D | 主文字 |
| `--red` | #D10000 | 几何构件 / 关键标题 / hover |
| `--red-bright` | #FF2200 | 亮红强调 |
| `--red-deep` | #8B0000 | 深红 |
| `--gray-mid` | #9A8B78 | 次要文字 |
| `--gray-light` | #C9B89A | 占位背景 |
| `--gray-dark` | #3D3228 | 暖黑色文字 |

### 使用法则

- 红色只出现在：几何色块、关键标题、hover 状态、分割线。
- 暖纸色为全局背景，黑色为主文字。
- 灰色（暖棕）用于辅助线和次要信息，但尽量少用——构成主义讨厌暧昧。
- 暗色模式：背景反转深棕，文字变暖白，红色不变。

## 字体

| 角色 | 字体 | CSS Variable |
|------|------|-------------|
| 中文标题（巨大） | 系统黑体栈（见下方） | `--font-display` |
| 正文 / 次级标题 | Geist Sans | `--font-geist-sans` |
| 标签 / 编号 / 数据 | Geist Mono | `--font-geist-mono` |

### 中文标题字体栈

```css
--font-display: "PingFang SC", "Noto Sans SC", "Microsoft YaHei", "Heiti SC", sans-serif;
```

原因：Geist Sans 的中文渲染不足以撑起 Rodchenko 级别的巨大标题。需要优质中文字体。

### 排版法则

- **极端尺度对比**：一个页面可以同时出现 `text-[12rem]` 的标题和 `text-xs` 的标签。
- **文字溢出屏幕是故意的**：标题被裁剪、被几何图形打断、半隐藏，都是设计意图。
- **中英文混排**：中文标题 + 俄文/英文小字标注（如「摄影」+「ФОТО」）。
- **全大写 + 宽字距**用于英文/拼音短标签。
- **粗体优先**，细体几乎不出现。

**详细规格**：

- **构图**：对角线分割。左下方 60% 区域被一个巨大的红色几何块占据，中文标题「关于我的一切」嵌在其中，白色大字，`font-size: clamp(4rem, 8vw, 9rem)`，`font-weight: 900`，字间距紧凑。

- **几何构件**：
  - 左上角一个黑色粗横条（高约 6px，宽 30%），工业感。

- **俄文标注**：小号西里尔字母 `ВСЁ ОБО МНЕ` 横排在红色块边缘，`letter-spacing: 0.3em`，灰色。
- **身份标签**：「学生 · 摄影师 · 影像创作者」用 Geist Mono，`text-sm`，放在红色块下方，黑色。
- **滚动指示**：底部中央一个向下的红色箭头（纯几何三角，不要圆角），像海报翻页。

### 海报 ③ — 「摄影档案」

**构成逻辑**：Rodchenko 式摄影蒙太奇

```
┌──────────────────────────────────────┐
│                                      │
│          ┌────┐                      │
│     ╱    │    │   ╱                  │
│    ╱     └────┘  ╱   [照片2]        │
│   ╱  [照片1]    ╱                    │
│  ╱             ╱                     │
│ ╱   ФОТО      ╱    ┌────┐           │
│     摄影档案       │    │           │
│                    └────┘           │
│                [照片3]              │
│                                    │
│   胶片 · 数码                       │
│   ─────────────                    │
│                                    │
└────────────────────────────────────┘
```

**详细规格**：

- **构图**：三条对角线（`skewY` 或 `rotate`）贯穿画面，将空间分成 4-5 个不规则的平行四边形区域。照片被放置在这些区域中，彼此重叠、交错。
- **主标题**：「摄影档案」巨大，`font-size: clamp(5rem, 10vw, 11rem)`，`font-weight: 900`，黑色。位于画面中央偏左。部分被一个悬浮的红色矩形几何块遮挡。
- **俄文标注**：`ФОТО` 使用 `skewX(-15deg)` 倾斜，`font-size: 3rem`，红色，放在标题上方。
- **照片布局**（三张代表作，不对齐，不统一尺寸）：
  - 照片 1：左上方，`rotate(-6deg)`，`clip-path: polygon(...)` 切掉一个角。
  - 照片 2：右上方，`rotate(4deg)`，略小于照片 1。
  - 照片 3：右下方，`rotate(-3deg)`，被一道红色粗线（6px）水平穿过。
- **几何构件**：
  - 一条红色对角线（宽 4px）从左上角延伸至画面中央。
  - 几个黑色小方块散布在空白处，大小不一。
  - 一个红色半透明矩形覆盖在照片交叠处，`opacity: 0.3`。
- **分类入口**（胶片 / 数码）：
  - 不放在卡片里。
  - 「胶片」和「数码」作为两个巨大的文字链接，排列在画面左下角，左右排列，中间用红色竖线（`|`）分隔。
  - hover 时红色竖线变成红色方块，文字变为红色。
  - 点击进入对应子页面。
- **子分类提示**：在「胶片」下方用小号 Mono 字体列出「纪实 · 创意」；同理「数码」下方列出「在场 · 纪实 · 创意」。

**尺寸与位置**：

| 元素 | 位置 | 尺寸 |
|------|------|------|
| 主标题「摄影档案」 | left: 12%, top: 38% | — |
| 俄文 ФОТО | left: 10%, top: 28% | — |
| 照片 1 | left: 8%, top: 8% | w: 35%, h: auto |
| 照片 2 | right: 6%, top: 14% | w: 28%, h: auto |
| 照片 3 | right: 18%, bottom: 18% | w: 32%, h: auto |

## 全局设计系统

### 几何构件库

这些元素在整个网站中复现：

| 构件 | 规格 | 用途 |
|------|------|------|
| 粗横线 | 3-6px 高，红或黑 | 分割、强调、海报感 |
| 对角线 | 2-4px 宽，红色，`skewY` | Rodchenko 式空间切割 |
| 大圆 | Ø 100-300px，红或黑 | Lissitzky 式悬浮几何 |
| 小方块 | 10-30px，红或黑 | 散落构图装饰 |
| 平行斜线组 | 3 根，间距 8px，`skewY(-30deg)` | 经典构成主义纹理 |
| 三角箭头 | 纯 CSS 三角（`border` hack） | 导航、滚动指示 |

### 动效系统

**原则**：所有动效必须是机械的、直接的、没有任何缓动曲线柔化。

- **transition-timing-function**：不使用 `ease` 或 `ease-in-out`。使用 `cubic-bezier(0.2, 0, 0, 1)` 或直接 `linear`。
- **hover 效果**：
  - 几何线 → 延长或变粗。
  - 文字 → 颜色从黑变红，或从白变红。
  - 图片 → `filter: grayscale(0)` → `grayscale(0.5)` 或反转。
  - 圆 → 旋转 45° 或位移 10px。
- **入场动画**（scroll-triggered，Intersection Observer）：
  - 几何线从画面边缘滑入（`translateX` / `translateY`）。
  - 标题从底部快速升起（`translateY(60px)` → `0`，duration 0.4s）。
  - 照片蒙太奇各块以轻微时差出现（stagger 0.1s）。
  - 圆形 scale 从 0.85 → 1（不是弹性的，是直接的）。
- **禁止**：fade-in-up 的软动画、弹簧动画、ease-in-out、blur 过渡。Rodchenko 的东西是锋利的。

### 暗色模式

- 背景：`#4A3F35`（深棕暖调）
- 文字：`#F0E6D6`（暖白）
- 红保持不变
- 图片可能需要略微提亮
- 几何构件颜色反转（暖黑→暖白，保留红色）

### 响应式策略

- 在大屏（≥1440px）：完整海报构图，所有几何元素到位。
- 在平板（768-1439px）：保持对角线逻辑，但缩小极端字号，减少几何元素密度。
- 在手机（<768px）：
  - 对角线构图改为纵向堆叠，但仍保留倾斜标题和几何元素。
  - 每个海报高度从 100vh 调整为 `auto`（内容自适应）。
  - 大字缩小但不失冲击力（`clamp(2.5rem, 12vw, 5rem)`）。
  - 圆形、斜线、方块保留但减少数量。
  - scroll-snap 关闭（移动端手感不佳）。
